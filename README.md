## Installation

To run this project locally, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/my-portfolio.git
    ```

2. Navigate to the project directory:
    ```bash
    cd my-portfolio
    ```

3. Install the dependencies:
    ```bash
    npm install
    ```
